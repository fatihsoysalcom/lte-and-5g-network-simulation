# LTE and 5G Network Simulation

This Python script simulates and compares the download times and latencies of LTE and 5G network technologies. It highlights how 5G offers significantly better performance while acknowledging LTE's continued relevance.

## Language

`python`

## How to Run

Run the script using: python main.py
